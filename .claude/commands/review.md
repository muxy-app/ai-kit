## Steps

1. Review the changes to fullfill the rules in the CLAUDE.md.
2. If everything is good, Approve
3. If changes are required/recommended, categories the changes in Critical, Warnings and Nits
4. Share the required/recommended changes out of the categories
5. Ask questions if you need user's input
6. Apply the recommendations if user agrees to
7. Apply the recommendations by sticking to the rules of the CLAUDE.md in a way that if I ask another agent for review, We don't get required changes.
8. When finished, Share guides with the user what to test
9. When confirmed the fix working, Share a PR title and very short PR description

## Guides

- Keep the communication to the user as short as possible with less reads
- Use question tool if you need to ask questions
